Hackathon Projects

Microsoft Hackathon – Legal Document Analyzer

At the Microsoft Redmond Hackathon, I worked on a legal document analysis tool designed to parse long contracts and highlight relevant clauses based on user queries. Our goal was to reduce the time it takes to manually sift through legal agreements by automating clause extraction and section summarization. I was responsible for designing the backend logic, where I built a pipeline that accepted multi-page PDFs, parsed the text using pdfminer.six, and organized content by detecting section headers, clause numbering patterns, and indentation heuristics. I implemented a custom rule-based segmentation algorithm that identified legal sections like “Termination,” “Governing Law,” or “Indemnification” based on regex patterns and phrase similarity. After segmentation, I used a lightweight NLP model (TF-IDF + cosine similarity) to match user queries against clause content, so users could retrieve the most relevant sections with contextual snippets. To make the system interactive, I exposed these components through a FastAPI backend and connected it to a basic frontend where users could upload documents, type queries, and get real-time responses. The end-to-end system reduced the time needed to locate legal content by over 70% in demo use cases. I also implemented basic caching to avoid redundant parsing and used Spacy for named entity recognition (e.g., contract dates, parties) to enhance contextual understanding. This project gave me hands-on experience combining document parsing, NLP, and full-stack deployment under tight time constraints.

Qualcomm x Northeastern Hackathon – AI Stylist

During the Qualcomm x Northeastern Hackathon, I developed an AI-powered wardrobe assistant that recommends weather-appropriate outfits by analyzing clothing images and environmental context. The goal was to combine computer vision and user interaction to deliver personalized style suggestions in real-time. My contribution focused on the ML backend. I used Qualcomm’s implementation of the DeepLabV3-ResNet50 model to segment clothing items from user-uploaded photos. I processed the model’s segmentation masks to isolate clothing regions, then mapped them to clothing categories (e.g., tops, pants) using post-processing heuristics based on bounding box area, aspect ratio, and color clustering. Once segmented, I linked each identified clothing item to a metadata tag (e.g., fabric weight, sleeve length) that could be filtered using weather conditions. I used the OpenWeatherMap API to fetch real-time temperature and weather data for the user's location. Based on these values, I applied filtering logic (e.g., exclude heavy jackets in warm weather, prioritize breathable fabrics when humid) to generate relevant outfit combinations. All of this logic ran on a FastAPI backend, and I stored processed user images and clothing masks on AWS S3. On the frontend, I built an interactive React interface that allowed users to drag and rearrange clothing items and save preferred combinations. The system demonstrated a full ML-integrated workflow — from image segmentation and filtering logic to cloud storage and user interaction — and it laid the foundation for building deeper personalization using recommendation algorithms.

Age and Gender Analysis Using Machine Learning

For a hackathon that happened during my undergraduation , I developed a machine learning system that predicts a person’s age range and gender based on facial images. The project was motivated by use cases in crowd analytics, demographic estimation, and user profiling for offline businesses. The first step involved constructing a labeled dataset using publicly available facial image repositories, such as UTKFace and Adience. I preprocessed these images using OpenCV to normalize brightness, scale face dimensions, and align facial landmarks. I then extracted facial embeddings using a pre-trained convolutional neural network (VGG16) and fine-tuned a smaller CNN model on our dataset to predict both age (in bins) and gender as separate outputs in a multi-task learning setup. For training, I used cross-entropy loss for gender classification and mean absolute error for age regression, applying dropout and batch normalization to avoid overfitting. I experimented with several optimizer configurations (Adam, SGD with momentum) and adjusted learning rates using ReduceLROnPlateau to improve convergence. On evaluation, the model achieved around 89% gender classification accuracy and an average error of 4.2 years on age estimation. I packaged the model using Flask and built a simple web interface that allowed users to upload an image and view predicted age/gender in real-time. This project deepened my understanding of computer vision pipelines, CNN architectures, and the importance of preprocessing and augmentation in real-world deployment of ML models.

I integrated these with AWS Secrets Manager to securely handle credential rotation and with Amazon SNS for push notifications during job failures.
